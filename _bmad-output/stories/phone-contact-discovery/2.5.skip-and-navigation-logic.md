# Story 2.5: Skip and Navigation Logic

## Status

Complete

## Story

**As a** new user who doesn't want to find friends now,
**I want** to skip the friend discovery step,
**So that** I can start using the app immediately.

## Acceptance Criteria

1. Close (X) button visible in top right corner of the Friend Discovery screen header
2. Tapping close (X) button:
   - Navigates to Home screen using `router.replace('/')`
   - Does NOT set any skip flag (user can still see friend discovery in profile later)
3. Back button/gesture navigates to Home (cannot go back to registration)
4. Closing the phone BottomSheet without entering phone:
   - User stays on Friend Discovery screen
   - Can close via close (X) button to proceed

## Dev Notes

### Navigation Pattern

```typescript
import { useRouter } from 'expo-router';

const FriendDiscoveryContainer = () => {
  const router = useRouter();

  const handleClose = () => {
    router.replace('/');
  };

  // Prevent back navigation
  useFocusEffect(
    useCallback(() => {
      const onBackPress = () => {
        router.replace('/');
        return true; // Prevent default back behavior
      };

      BackHandler.addEventListener('hardwareBackPress', onBackPress);
      return () => BackHandler.removeEventListener('hardwareBackPress', onBackPress);
    }, [router])
  );

  return (
    <SafeAreaView>
      <Header>
        <Title>Retrouve tes amis</Title>
        <CloseButton onPress={handleClose} />
      </Header>
      {/* ... content ... */}
    </SafeAreaView>
  );
};
```
