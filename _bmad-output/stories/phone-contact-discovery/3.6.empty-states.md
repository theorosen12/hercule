# Story 3.6: Empty States for Each Section

## Status

Draft

## Story

**As a** new user on the Friend Discovery screen,
**I want** to see appropriate empty states for each section,
**So that** I understand when no contacts are available.

## Acceptance Criteria

1. Section "Amis sur Nara" empty state: "Aucun ami trouvé sur Nara"
2. Section "Inviter des contacts": hidden entirely if no device contacts
3. Both sections empty: show combined message with generic invite option
4. Permission denied: show message for both sections

- [ ] **Task 1: Create SectionEmptyState Component** (AC: 1)
  - [ ] Create reusable empty state component for sections
  - [ ] Accept message and optional action button
  - [ ] Style with subtle icon and centered text

- [ ] **Task 2: Implement Suggestions Empty State** (AC: 1)

- [ ] **Task 3: Handle Both Sections Empty** (AC: 3)

### SectionEmptyState Component

```typescript
// components/SectionEmptyState/SectionEmptyState.component.tsx
import React from 'react';
import { useIntl } from 'react-intl';

import { Typography } from '@nara/evanescent/components/Display/Typography/Typography.component';
import { Button } from '@nara/evanescent/components/Input/Button/Button.component';

import * as S from './SectionEmptyState.styles';

interface SectionEmptyStateProps {
  message: string;
  icon?: React.ReactNode;
  actionLabel?: string;
  onAction?: () => void;
  testID?: string;
}

export const SectionEmptyState: React.FC<SectionEmptyStateProps> = ({
  message,
  icon,
  actionLabel,
  onAction,
  testID,
}) => {
  return (
    <S.Container testID={testID}>
      {icon && <S.IconContainer>{icon}</S.IconContainer>}
      <S.MessageText>{message}</S.MessageText>
      {actionLabel && onAction && (
        <S.ActionContainer>
          <Button
            label={actionLabel}
            onPress={onAction}
            variant="secondary"
            size="small"
          />
        </S.ActionContainer>
      )}
    </S.Container>
  );
};
```

### Updated FriendDiscovery Container

```typescript
// FriendDiscovery.container.tsx - renderContent updates

const hasSuggestions = suggestions.length > 0;
const hasDeviceContacts = deviceContacts.length > 0;
const hasAnyContacts = hasSuggestions || hasDeviceContacts;

const handleGenericInvite = useCallback(() => {
  Share.share({
    message: intl.formatMessage(friendDiscoveryMessages.genericInviteMessage, {
      url: INVITE_URL,
    }),
  });
}, [intl]);

const handleOpenSettings = useCallback(() => {
  Linking.openSettings();
}, []);

// In render:
const renderContent = () => {
  // Permission denied state
  if (discoveryState === 'denied') {
    return (
      <SectionEmptyState
        message={intl.formatMessage(friendDiscoveryMessages.permissionDeniedMessage)}
        actionLabel={intl.formatMessage(friendDiscoveryMessages.openSettingsButton)}
        onAction={handleOpenSettings}
        testID="permission-denied-state"
      />
    );
  }

  // Both sections empty
  if (!hasAnyContacts && discoveryState !== 'loading') {
    return (
      <SectionEmptyState
        message={intl.formatMessage(friendDiscoveryMessages.noContactsAtAll)}
        actionLabel={intl.formatMessage(friendDiscoveryMessages.inviteFriendsButton)}
        onAction={handleGenericInvite}
        testID="no-contacts-state"
      />
    );
  }

  // Normal two-section layout
  return (
    <>
      {/* Section 1: Amis sur Nara */}
      <S.Section>
        <SectionHeader title="Amis sur Nara" count={suggestions.length} />
        {hasSuggestions ? (
          <PaginatedList
            items={suggestions}
            pageSize={3}
            keyExtractor={(item) => item.id}
            renderItem={(suggestion) => (
              <SuggestionRow
                suggestion={suggestion}
                onAddFriend={handleAddFriend}
                requestStatus={requestStatuses.get(suggestion.id)}
              />
            )}
          />
        ) : (
          <SectionEmptyState
            message={intl.formatMessage(friendDiscoveryMessages.noSuggestionsFound)}
          />
        )}
      </S.Section>

      {/* Section 2: Inviter des contacts */}
      {hasDeviceContacts && (
        <S.Section>
          <SectionHeader title="Inviter des contacts" count={deviceContacts.length} />
          <PaginatedList
            items={deviceContacts}
            pageSize={3}
            keyExtractor={(item) => item.phoneNumber}
            renderItem={(contact) => (
              <InviteContactRow contact={contact} />
            )}
          />
        </S.Section>
      )}
    </>
  );
};
```
