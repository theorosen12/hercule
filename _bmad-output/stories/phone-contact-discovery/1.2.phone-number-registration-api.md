# Story 1.2: Phone Number Registration API

## Status

Done

## Story

**As a** mobile app user,
**I want** to register my phone number,
**so that** my friends can discover me on the platform.

## Acceptance Criteria

1. Integrate `libphonenumber-js` for E.164 normalization
2. Implement HMAC-SHA256 hashing with server-side secret key (`PHONE_HASH_SECRET` env var)
3. Create `setMyPhoneNumber(phoneNumber: String!): User!` mutation for user phone registration
4. Validate phone number format, reject invalid with clear error
5. Hash and store user's phone number in `HashedPhoneNumber` table
6. Require JWT authentication
7. Never log plain text phone numbers

**PhoneHashingService Implementation:**
```typescript
import { Injectable } from '@nestjs/common';
import { ConfigService } from '@nestjs/config';
import { parsePhoneNumber, isValidPhoneNumber } from 'libphonenumber-js';
import * as crypto from 'crypto';

@Injectable()
export class PhoneHashingService {
  private readonly hashSecret: string;

  constructor(private readonly configService: ConfigService) {
    this.hashSecret = this.configService.get<string>('PHONE_HASH_SECRET')
      || 'dev-phone-hash-secret-change-in-production';
  }

  normalizePhoneNumber(rawPhone: string): string | null {
    try {
      const parsed = parsePhoneNumber(rawPhone, 'FR'); // Default country
      if (!parsed.isValid()) {
        return null;
      }
      return parsed.format('E.164'); // e.g., "+33612345678"
    } catch {
      return null;
    }
  }

  validatePhoneNumber(rawPhone: string): boolean {
    try {
      return isValidPhoneNumber(rawPhone, 'FR');
    } catch {
      return false;
    }
  }

  hashPhoneNumber(e164Phone: string): string {
    return crypto
      .createHmac('sha256', this.hashSecret)
      .update(e164Phone)
      .digest('hex');
  }

  normalizeAndHash(rawPhone: string): string | null {
    const normalized = this.normalizePhoneNumber(rawPhone);
    if (!normalized) {
      return null;
    }
    return this.hashPhoneNumber(normalized);
  }
}
```

**Resolver Implementation:**
```typescript
@Resolver(() => User)
export class ContactResolver {
  constructor(
    private readonly phoneHashingService: PhoneHashingService,
    private readonly prisma: PrismaService,
    private readonly logger: LoggerService,
  ) {}

  @Mutation(() => User)
  @UseGuards(JwtAuthGuard)
  @Throttle({ default: { limit: 3, ttl: 86400 } })
  async setMyPhoneNumber(
    @Args('phoneNumber') phoneNumber: string,
    @CurrentUser() user: User,
  ): Promise<User> {
    // 1. Validate and normalize
    const normalized = this.phoneHashingService.normalizePhoneNumber(phoneNumber);
    if (!normalized) {
      throw new PhoneNumberInvalidException();
    }

    // 2. Hash (synchronous with HMAC-SHA256)
    const hashedPhone = this.phoneHashingService.hashPhoneNumber(normalized);

    // 3. Upsert HashedPhoneNumber
    try {
      await this.prisma.hashedPhoneNumber.upsert({
        where: { userId: user.id },
        update: { hashedPhone },
        create: { userId: user.id, hashedPhone },
      });
    } catch (error) {
      if (error.code === 'P2002') {
        throw new PhoneNumberInUseException();
      }
      throw error;
    }

    // 4. Audit log (no PII)
    this.logger.log({
      event: 'PHONE_REGISTERED',
      userId: user.id,
      timestamp: new Date().toISOString(),
    });

    return user;
  }
}
```
