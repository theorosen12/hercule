# Story 2.3: Contact Permission and Suggestions Display

## Status

Complete

## Story

**As a** new user who entered my phone number,
**I want** to see friend suggestions on the page after granting contact permission,
**So that** I can quickly connect with friends.

## Acceptance Criteria

1. After phone number saved (BottomSheet closes), immediately request contact permission via `expo-contacts`
2. If permission **granted**:
   - Extract phone numbers from contacts
   - Call `getContactSuggestions` query
   - Display suggestions using `SuggestionList` component directly on the Friend Discovery page
   - Show `NoSuggestionsState` if no matches found
3. If permission **denied**:
   - Display friendly message on page: "Tu pourras retrouver tes amis plus tard dans ton profil"
   - User can close via the close (X) button in header to proceed to Home
4. Show loading state during contact sync and query
5. Show appropriate state transitions on the page (loading → suggestions OR permission denied)

## Dev Notes

### Flow Sequence

```
Phone Entry BottomSheet (2.2)
    ↓ [Phone Saved & BottomSheet Closes]
Contact Permission Request (System Dialog)
    ├─→ Granted
    │       ↓
    │   Extract Contacts → Query Suggestions
    │       ├─→ Has suggestions → Show SuggestionList on page
    │       └─→ No suggestions → Show NoSuggestionsState on page
    │
    └─→ Denied → Show "retrouver plus tard" message on page
```

### Integration in FriendDiscovery Container

```typescript
const FriendDiscoveryContainer = () => {
  const { user } = useCurrentUserContext();
  const [isBottomSheetOpen, setIsBottomSheetOpen] = useState(!user.hasPhoneNumber);
  const [suggestions, setSuggestions] = useState<ContactSuggestion[]>([]);
  const [permissionState, setPermissionState] = useState<'pending' | 'granted' | 'denied'>('pending');
  const [isLoading, setIsLoading] = useState(false);

  const { requestPermission, extractContacts } = useContactPermissionFlow();
  const [getContactSuggestions] = useGetContactSuggestionsLazyQuery();

  const handlePhoneSuccess = async () => {
    setIsBottomSheetOpen(false);
    setIsLoading(true);

    const permission = await requestPermission();
    if (permission === 'granted') {
      setPermissionState('granted');
      const contacts = await extractContacts();
      const phoneNumbers = extractPhoneNumbers(contacts);
      const result = await getContactSuggestions({ variables: { phoneNumbers } });
      setSuggestions(result.data?.getContactSuggestions ?? []);
    } else {
      setPermissionState('denied');
    }

    setIsLoading(false);
  };

  const renderContent = () => {
    if (isLoading) {
      return <LoadingState />;
    }

    if (permissionState === 'denied') {
      return <PermissionDeniedMessage />;
    }

    if (suggestions.length > 0) {
      return <SuggestionList suggestions={suggestions} />;
    }

    if (permissionState === 'granted') {
      return <NoSuggestionsState />;
    }

    return <InitialState />;
  };

  return (
    <SafeAreaView>
      <Header>
        <Title>Retrouve tes amis</Title>
        <CloseButton onPress={handleClose} />
      </Header>
      <ContentArea>{renderContent()}</ContentArea>
      <PhoneEntryBottomSheet
        isOpen={isBottomSheetOpen}
        onClose={() => setIsBottomSheetOpen(false)}
        onSuccess={handlePhoneSuccess}
      />
    </SafeAreaView>
  );
};
```
