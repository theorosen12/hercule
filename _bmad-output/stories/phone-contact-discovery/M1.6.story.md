# Story M1.6: No Suggestions State

## Status: Done

## Story

**As a** user whose contacts don't match any platform users,
**I want** to see a helpful message with an invite option,
**so that** I can bring my friends to the platform.

2. Display message: "Tes amis ne sont pas encore sur Nara !" and then "Mais tu peux les inviter à te rejoindre pour raconter et revivre vos souvenirs ensemble !"
3. Display line with user card then button on the right with "Inviter des amis" button with Gift Icon

## Implementation

```typescript
// NoSuggestionsState.component.tsx
import { Share, Alert } from 'react-native';
import { useIntl } from 'react-intl';
import { noSuggestionsMessages } from './NoSuggestionsState.messages';
import * as S from './NoSuggestionsState.styles';

const INVITE_LINK = 'https://app.example.com/invite'; // TODO: Get from config

export const NoSuggestionsState = () => {
  const intl = useIntl();

  const handleInvite = async () => {
    try {
      await Share.share({
        message: intl.formatMessage(noSuggestionsMessages.shareMessage, {
          link: INVITE_LINK,
        }),
        // iOS only
        url: INVITE_LINK,
      });
    } catch (error) {
      if (error instanceof Error && error.message !== 'User cancelled') {
        Alert.alert(
          intl.formatMessage(noSuggestionsMessages.shareError)
        );
      }
    }
  };

  return (
    <S.Container>
      <S.Illustration source={require('./assets/no-friends.png')} />

      <S.Title>
        {intl.formatMessage(noSuggestionsMessages.title)}
      </S.Title>

      <S.Description>
        {intl.formatMessage(noSuggestionsMessages.description)}
      </S.Description>

      <S.InviteButton onPress={handleInvite}>
        <S.InviteButtonText>
          {intl.formatMessage(noSuggestionsMessages.inviteButton)}
        </S.InviteButtonText>
      </S.InviteButton>
    </S.Container>
  );
};
```
