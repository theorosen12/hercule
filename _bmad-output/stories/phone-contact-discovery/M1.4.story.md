# Story M1.4: Suggestions List Display

## Status: Ready for Review

## Story

**As a** user who granted contact permission,
**I want** to see my friend suggestions in a vertical list,
**so that** I can quickly browse and add friends.

## Architecture Note (v3.1)

Uses `getContactSuggestions(phoneNumbers)` query - contacts passed as parameter, never stored.

## Acceptance Criteria

1. Create `SuggestionList` component with vertical `FlatList`
2. Each `SuggestionRow` shows:
   - Profile picture (or default avatar) on the left
   - First name + Last name
   - "Ajouter" button with `PlusCircle` icon on the right
3. Create `getContactSuggestions.query.gql` in `src/domains/user/contactDiscovery/infra/queries/`
4. Run `npm run codegen` to generate hooks
5. Show loading skeleton while fetching suggestions
6. List scrolls smoothly (60fps)

## Technical Notes

- Use standard vertical `FlatList` for performance
- Each row is a horizontal layout with avatar, name, and add button
- Use `PlusCircle` icon from the design system (lucide-react-native or equivalent)

## GraphQL Query

```graphql
# src/domains/user/contactDiscovery/infra/queries/getContactSuggestions.query.gql

query GetContactSuggestions($phoneNumbers: [String!]!) {
  getContactSuggestions(phoneNumbers: $phoneNumbers) {
    id
    username
    firstName
    lastName
    profilePictureMediaId
  }
}
```

**Note:** Phone numbers are passed as query parameter. The query is typically called from M1.3 (useContactDiscovery hook) which extracts contacts and passes them to this query.

## Implementation

```typescript
// SuggestionList.component.tsx
import { FlatList } from 'react-native';
import { SuggestionRow } from '../SuggestionRow';

interface Props {
  suggestions: User[];
  onAddFriend: (userId: string) => void;
  pendingRequests: Set<string>;
}

export const SuggestionList = ({ suggestions, onAddFriend, pendingRequests }: Props) => {
  return (
    <FlatList
      data={suggestions}
      keyExtractor={(item) => item.id}
      renderItem={({ item }) => (
        <SuggestionRow
          user={item}
          onAdd={() => onAddFriend(item.id)}
          isPending={pendingRequests.has(item.id)}
        />
      )}
      contentContainerStyle={styles.container}
      ItemSeparatorComponent={Separator}
    />
  );
};
```

```typescript
// SuggestionRow.component.tsx
import { PlusCircle } from 'lucide-react-native';

interface Props {
  user: {
    id: string;
    firstName: string;
    lastName: string;
    profilePictureMediaId?: string | null;
  };
  onAdd: () => void;
  isPending: boolean;
}

export const SuggestionRow = ({ user, onAdd, isPending }: Props) => {
  return (
    <Row>
      <LeftSection>
        <ProfilePicture mediaId={user.profilePictureMediaId} size={48} />
        <Name>{user.firstName} {user.lastName}</Name>
      </LeftSection>
      {isPending ? (
        <PendingLabel>En attente</PendingLabel>
      ) : (
        <AddButton onPress={onAdd}>
          <PlusCircle size={20} />
          <ButtonText>Ajouter</ButtonText>
        </AddButton>
      )}
    </Row>
  );
};
```
