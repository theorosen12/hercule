# Story 3.4: Display Narative Count on Suggestion Row

## Status

Draft

## Story

**As a** new user viewing a friend suggestion,
**I want** to see how many Naratives the person is member of,
**So that** I can gauge how active they are on the platform.

## Acceptance Criteria

1. Display narative count on `SuggestionRow` component
2. Format: "{count} Nara" or "{count} Naras" (pluralization)
3. Show "Nouveau" badge if count is 0 (new user)
4. Position: below the user's name
5. Use secondary text styling

## Dev Notes

### Updated SuggestionRow Component

```typescript
// SuggestionRow.component.tsx
interface SuggestionRowProps {
  suggestion: ContactSuggestion;
  onAddFriend: (userId: string) => void;
  requestStatus?: FriendRequestStatus;
  testID?: string;
}

export const SuggestionRow: React.FC<SuggestionRowProps> = ({
  suggestion,
  onAddFriend,
  requestStatus,
  testID,
}) => {
  const intl = useIntl();

  const renderNarativeCount = () => {
    if (suggestion.narativeCount === 0) {
      return (
        <S.NewBadge>
          {intl.formatMessage(messages.newUser)}
        </S.NewBadge>
      );
    }

    return (
      <S.NarativeCountText>
        {intl.formatMessage(messages.narativeCount, {
          count: suggestion.narativeCount,
        })}
      </S.NarativeCountText>
    );
  };

  return (
    <S.Container testID={testID}>
      <S.AvatarContainer>
        <Avatar ... />
      </S.AvatarContainer>

      <S.InfoContainer>
        <S.NameText>
          {suggestion.firstName} {suggestion.lastName}
        </S.NameText>
        {renderNarativeCount()}
      </S.InfoContainer>

      <S.ActionContainer>
        <AddFriendButton ... />
      </S.ActionContainer>
    </S.Container>
  );
};
```
