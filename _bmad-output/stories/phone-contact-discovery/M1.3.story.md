# Story M1.3: Contact Permission and Friend Discovery

## Status: Done

## Story

**As a** user who just entered my phone number,
**I want** the app to automatically request contact permission and find friends,
**so that** I can see friend suggestions without extra steps.

## Architecture Note (v3.1)

This story uses the **stateless architecture** where contacts are sent directly to `getContactSuggestions(phoneNumbers)` query - no `uploadContacts` mutation needed.

## Acceptance Criteria

1. After phone number is saved, immediately call `expo-contacts.requestPermissionsAsync()`
2. If permission granted:
   - Extract phone numbers via `getContactsAsync({ fields: [PhoneNumbers] })`
   - Deduplicate extracted phone numbers
   - Call `getContactSuggestions(phoneNumbers)` query directly (contacts as parameter)
   - Display suggestions (no separate upload step)
3. If permission denied:
   - Hide the module for current session
   - Module reappears on next app session (no persistent flag)
4. Show loading state during discovery process
5. Handle errors gracefully (network, rate limit)

## Technical Notes

- **Do NOT** show explanation dialog before permission - request directly
- **Session-based hiding** for permission denial (no AsyncStorage)
- **Contacts are NOT stored** - sent directly to query, matched at runtime

## Implementation

```typescript
// useContactDiscovery.hook.ts
import * as Contacts from 'expo-contacts';
import { useGetContactSuggestionsLazyQuery } from '@generated/graphql';

export const useContactDiscovery = () => {
  const [getSuggestions, { data, loading, error }] = useGetContactSuggestionsLazyQuery();
  const [sessionHidden, setSessionHidden] = useState(false);

  const discoverFriends = async () => {
    try {
      // 1. Request permission
      const { status } = await Contacts.requestPermissionsAsync();

      if (status !== 'granted') {
        setSessionHidden(true);
        return { success: false, reason: 'permission_denied' };
      }

      // 2. Extract phone numbers
      const { data: contactsData } = await Contacts.getContactsAsync({
        fields: [Contacts.Fields.PhoneNumbers],
      });

      const phoneNumbers = extractAndDedupePhones(contactsData);

      // 3. Query suggestions directly (contacts as parameter - NOT stored)
      const result = await getSuggestions({
        variables: { phoneNumbers },
      });

      return { success: true, suggestions: result.data?.getContactSuggestions };

    } catch (error) {
      return { success: false, error };
    }
  };

  return {
    discoverFriends,
    suggestions: data?.getContactSuggestions ?? [],
    isLoading: loading,
    error,
    sessionHidden
  };
};

const extractAndDedupePhones = (contacts: Contacts.Contact[]): string[] => {
  const phones = new Set<string>();

  contacts.forEach((contact) => {
    contact.phoneNumbers?.forEach((phone) => {
      if (phone.number) {
        phones.add(phone.number);
      }
    });
  });

  return Array.from(phones);
};
```

## GraphQL Query

```graphql
# src/domains/user/contactDiscovery/infra/queries/getContactSuggestions.query.gql

query GetContactSuggestions($phoneNumbers: [String!]!) {
  getContactSuggestions(phoneNumbers: $phoneNumbers) {
    id
    username
    firstName
    lastName
    profilePictureMediaId
  }
}
```

## Error Handling

| Error | Behavior |
|-------|----------|
| Permission denied | Hide module for session |
| Network error | Show retry option |
| `RATE_LIMIT_EXCEEDED` | Show "Réessayez dans X minutes" |
| `CONTACT_LIMIT_EXCEEDED` | Truncate to 1000 contacts |

## Key Differences from v1.0

| Before (v1.0) | After (v3.1) |
|---------------|--------------|
| `uploadContacts` then `getContactSuggestions` | Single `getContactSuggestions(phoneNumbers)` call |
| Contacts stored in database | Contacts NEVER stored |
| Two network requests | One network request |
